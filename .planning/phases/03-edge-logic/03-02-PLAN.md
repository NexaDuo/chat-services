## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Internet → Cloudflare Worker | Public traffic enters the edge layer. |
| Cloudflare Worker → Middleware | Authenticated internal call for tenant resolution. |
| Cloudflare Worker → Origins | Proxied traffic with injected headers to internal services. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-01 | Spoofing | Header Injection | mitigate | Origin services must only trust requests coming from the Worker IP range or via a secure tunnel. |
| T-03-02 | Information Disclosure | Middleware API | mitigate | `/resolve-tenant` endpoint is protected by a 16+ char shared secret (`Bearer` token). |
| T-03-03 | Denial of Service | Resolution Latency | mitigate | Worker implements an in-memory cache for slug-to-ID resolution to reduce load on Middleware. |
</threat_model>
