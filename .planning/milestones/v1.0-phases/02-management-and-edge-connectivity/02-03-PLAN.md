<task type="auto">
  <name>Task 1: Configure GCS Destination in Coolify</name>
  <files></files>
  <action>
    - Log in to Coolify dashboard.
    - Go to Settings -> Backup Destinations.
    - Add "S3 Compatible" destination.
    - Endpoint: `https://storage.googleapis.com`.
    - Region: `auto`.
    - Access Key: [GCS_HMAC_ACCESS_ID].
    - Secret Key: [GCS_HMAC_SECRET].
    - Bucket: `nexaduo-coolify-backups`.
  </action>
  <verify>
    <automated>curl -f -s http://localhost:3000/api/v1/destinations | grep "nexaduo-coolify-backups"</automated>
  </verify>
  <done>GCS destination shows as connected/valid in Coolify.</done>
</task>

<task type="auto">
  <name>Task 2: Schedule Daily Backups & Rotation</name>
  <files></files>
  <action>
    - In Coolify Settings, navigate to the internal database / configuration backup section.
    - Select the GCS destination created in Task 1.
    - Set schedule: Daily (02:00 UTC).
    - Set retention policy: 30 days.
    - Trigger a "Backup Now" and verify successful upload to the GCS bucket.
  </action>
  <verify>
    <automated>gcloud storage ls gs://nexaduo-coolify-backups/**</automated>
  </verify>
  <done>Initial backup exists in GCS and schedule is active.</done>
</task>
